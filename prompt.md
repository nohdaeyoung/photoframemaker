# Prompt Log — Photo Frame Maker by 324.ing

개발 과정에서 사용된 주요 프롬프트를 일자별로 기록합니다.

---

## 2026-02-27

### 프로젝트 초기 생성
> 사진에 여백 프레임을 추가할 수 있는 웹 앱을 만들어줘.
> 캔버스 비율 선택, 프레임 두께/색상 조절, PNG 다운로드 기능이 필요해.

### 이미지 표시 모드 변경
> 이미지가 잘리지 않게 전체가 보이도록 해줘 (Contain 모드)

### UI 레이아웃 조정
> 설정 패널을 오른쪽으로 옮겨줘.
> 프리뷰 영역 클릭하면 사진 업로드되게 해줘.
> 사진 삭제 버튼 추가해줘.

### 타이틀 변경
> 타이틀을 "Photo Frame Maker by 324.ing"으로 바꿔줘

### 모바일 UX 개선
> 모바일에서 하단에 탭 바 형태로 바꿔줘.
> 모바일에서 터치가 안 되는 문제 해결해줘.
> 모바일 사진 탭에 썸네일이랑 파일 정보 보여줘.

### EXIF 메타데이터
> EXIF 정보를 파싱해서 정보 섹션에 보여줘 (카메라, 렌즈, 조리개, ISO 등)

### 다운로드 개선
> 다운로드 파일명을 원본이름_pfm.png로 해줘.
> iOS에서 사진 앱에 바로 저장되게 해줘 (Web Share API).

### 프리뷰 영역
> 프리뷰 영역의 체크무늬 배경이 캔버스 비율에 맞게 표시되도록 해줘.
