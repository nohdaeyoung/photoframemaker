## 2026-02-27

### 프로젝트 초기 생성
> 사진에 여백 프레임을 추가할 수 있는 웹 앱을 만들어줘.
> 캔버스 비율 선택, 프레임 두께/색상 조절, PNG 다운로드 기능이 필요해.

> 이미지가 잘리지 않게 전체가 보이도록 해줘 (Contain 모드)

### UI 레이아웃 조정
> 설정 패널을 오른쪽으로 옮겨줘.
> 프리뷰 영역 클릭하면 사진 업로드되게 해줘.
> 사진 삭제 버튼 추가해줘.

### 타이틀 변경
> 타이틀을 "Photo Frame Maker by 324.ing"으로 바꿔줘

### 모바일 UX 개선
> 모바일에서 하단에 탭 바 형태로 바꿔줘.
> 모바일에서 터치가 안 되는 문제 해결해줘.
> 모바일 사진 탭에 썸네일이랑 파일 정보 보여줘.

### EXIF 메타데이터
> EXIF 정보를 파싱해서 정보 섹션에 보여줘 (카메라, 렌즈, 조리개, ISO 등)

### 다운로드 개선
> 다운로드 파일명을 원본이름_pfm.png로 해줘.
> iOS에서 사진 앱에 바로 저장되게 해줘 (Web Share API).

### 프리뷰 영역
> 프리뷰 영역의 체크무늬 배경이 캔버스 비율에 맞게 표시되도록 해줘.

> GTM, GA4, 네이버 서치어드바이저 태그 추가해줘.

### About 페이지 분리
> 해당 내용은 /about 파일을 만들어서 거기다 넣고 하단 푸터에 링크시켜줘

### 다운로드 수정
> 데스크톱에서 다운로드 시 Share Sheet 대신 파일 직접 다운로드되게 해줘.

### 이미지 드래그 수정
> 이미지 드래그 위치조정 기능 수정해줘 (contain → cover).

### 썸네일 스트립 개선
> 썸네일 크기 25% 확대해줘.
> 썸네일 스트립 가운데 정렬 수정해줘.

### 인스타그램 다크모드
> 인스타그램 피드/프로필 미리보기에 다크모드 지원 추가해줘.

### 피드 모드 네비게이션
> 피드 영역에 2장 이상의 사진이 있을 경우 기본과 마찬가지로 좌우 화살표 버튼을 추가하여 다음/이전 사진으로 이동할 수 있는 기능 추가 (모바일은 슬라이드 적용)

### 프로필 모드 다중 이미지 배치
> 프로필 영역에 2장 이상의 사진이 있을 경우 1장일 때와는 다르게 첫번째칸 (좌상단)부터 사진이 적용되도록 변경

### Vercel 배포
> Vercel로 배포할 수 있게 vercel.json 설정해줘.

### 다중 이미지 업로드 안정화
> 다중 이미지 로딩 시 파일 순서가 섞이는 문제 수정해줘.
> 모바일에서 다중 이미지 업로드 시 레이아웃 깨지는 문제 수정해줘.

### 모바일 썸네일 UX 전면 개편
> 모바일 썸네일 5열 그리드 레이아웃으로 바꿔줘.
> 모바일 썸네일이 캔버스를 가리는 문제 수정해줘.
> 피드/프로필 모드에서 썸네일이 목업을 가리는 문제 수정해줘.
> 모바일 썸네일 목록을 '사진' 토글 버튼으로 접기/펼치기 해줘.
> 모바일 썸네일을 사진 탭 패널 안으로 이동해줘.
> 모바일 프리뷰 영역의 썸네일 중복 노출 제거해줘.

### 이미지 렌더링 모드 변경
> 이미지 렌더링을 contain 모드로 변경해줘 (사진 전체 표시).

### 네비게이션 속도 최적화
> 이미지 슬라이드 네비게이션 속도 최적화해줘.

### 인스타 비율 라벨 수정
> 인스타 게시물 라벨을 4:5에서 3:4로 이동해줘.
> 프로필 피드 썸네일 비율을 1:1에서 3:4로 변경해줘.

### 프리뷰 컨테이너 중앙 정렬 수정
> 세로 비율(9:16, 2:3) 선택 시 프리뷰 컨테이너가 중앙 정렬 안 되는 문제 수정해줘.
> 모바일에서 세로 비율 선택 시 왼쪽으로 치우치는 문제 수정해줘.

### 모바일 사진 탭 UX 개선
> 모바일에서 사진 없을 때 사진 탭 클릭하면 바로 파일 선택 실행되게 해줘.
