Photo Frame Maker란?

사진작가 DY가 만든 무료 온라인 사진 편집 도구 모음입니다. 포토샵 없이도 프레이밍, 분할, EXIF 오버레이, 포맷 변환, AI 업스케일링까지 — 사진에 필요한 모든 작업을 브라우저에서 바로 할 수 있습니다.

회원가입 없이 접속 즉시 사용하며, 모든 처리는 브라우저에서 이루어져 사진이 서버로 전송되지 않습니다. 광고도 없습니다.

Photo Frame Maker (f.324.ing) is a free online photo editing toolkit created by photographer Noh Daeyoung (DY). It includes five specialized tools: Frame Maker for adding borders and Instagram-style frames, Photo Splitter for grid splitting, EXIF Frame Designer for stylish metadata overlays, Image Converter for format conversion and resizing, and AI Upscaler for 4x enhancement using Real-ESRGAN. All processing runs locally in the browser — no photos are ever uploaded to a server.

5가지 도구

프레임 메이커

사진에 깔끔한 여백 프레임을 추가합니다. 기본 프레임, Instagram 피드, Instagram 프로필 3가지 모드를 제공하며, 7가지 캔버스 비율과 프레임 색상·두께를 자유롭게 조절할 수 있습니다.

사진 분할기

한 장의 사진을 2~4칸으로 분할합니다. 인스타그램 멀티 포스팅용 파노라마 분할에 최적화되어 있으며, 분할된 이미지를 일괄 다운로드할 수 있습니다.

EXIF 프레임 디자이너 New

사진의 촬영 정보(카메라, 렌즈, 조리개, ISO 등)를 아름다운 프레임으로 표현합니다. Fujifilm Recipe, Leica, Polaroid, Film Strip, Glass 등 18가지 스타일을 제공합니다.

이미지 변환기

JPG, PNG, WebP 포맷 간 상호 변환과 리사이즈를 지원합니다. 출력 품질을 직접 조절하여 용도에 맞는 최적의 파일을 만들 수 있습니다.

AI 사진 업스케일러 Beta

AI(Real-ESRGAN)를 활용해 사진을 4배 확대합니다. ONNX Runtime 기반 브라우저 내 AI 추론으로, 서버 전송 없이 고해상도 결과를 얻을 수 있습니다. Before/After 비교 슬라이더도 제공합니다.

주요 기능

  • 브라우저 로컬 처리 — 사진이 서버로 전송되지 않습니다. 개인정보 걱정 없이 사용하세요.
  • 가입·광고 없음 — 접속 즉시 사용. 방해 요소가 전혀 없습니다.
  • 7가지 캔버스 비율 — 1:1, 4:5, 3:4(인스타 게시물), 9:16(스토리/릴스), 4:3, 3:2, 2:3
  • Instagram 피드·프로필 프레이밍 — 좋아요, 팔로워 등 UI 요소가 포함된 목업 프레임
  • 18가지 EXIF 스타일 — Fujifilm Recipe, Leica, Polaroid, Film Strip, Glass, HUD, Editorial 등
  • AI 4배 업스케일 — Real-ESRGAN 모델로 저해상도 사진을 고해상도로 복원
  • 다중 이미지 처리 — 최대 10장까지 업로드, 동일 설정 일괄 적용, ZIP 다운로드
  • 무손실 PNG / 고화질 JPEG — 도구에 따라 최적의 출력 포맷 지원
  • 다크모드 — 시스템 설정에 따라 자동 전환
  • iOS 사진앱 직접 저장 — Web Share API로 사진앱에 바로 저장 (iOS Safari)

사용 방법

  1. 도구 선택 — 메인 페이지에서 프레임, 분할, EXIF 프레임, 변환, 업스케일 중 원하는 도구를 선택합니다.
  2. 사진 업로드 — 드래그 앤 드롭, 클릭, 또는 Ctrl+V로 붙여넣기 (프레임 메이커는 최대 10장)
  3. 설정 조정 — 캔버스 비율, 프레임 색상, EXIF 스타일, 출력 포맷 등 각 도구의 옵션을 조절합니다.
  4. 다운로드 — PNG, JPEG 또는 WebP로 저장. 다중 이미지는 ZIP으로 일괄 다운로드됩니다.

자주 묻는 질문

완전 무료인가요?
네, 모든 도구가 완전 무료이며 회원가입도 필요 없습니다.
사진이 서버에 업로드되나요?
아닙니다. AI 업스케일러를 포함한 모든 처리가 브라우저 내에서 이루어지며, 사진은 서버로 전송되지 않습니다.
어떤 이미지 포맷을 지원하나요?
JPG, PNG, WebP, GIF, BMP, AVIF를 입력으로 지원합니다. 출력은 도구에 따라 PNG, JPEG, WebP를 지원합니다.
EXIF 프레임 디자이너는 어떤 스타일을 제공하나요?
Film Strip, Minimal, Magazine, Signature, Letterbox, Polaroid, Leica, Fuji Style, Fuji Recipe, Glass, Leica Lux, Instax, Film Stock, Shot On, Editorial, HUD, Minimal Bar, Card Grid 총 18가지 스타일을 제공합니다.
AI 업스케일러는 어떤 기술을 사용하나요?
Real-ESRGAN 모델을 ONNX Runtime으로 브라우저에서 실행합니다. GPU 가속을 지원하며, 사진을 4배 확대합니다.
모바일에서도 사용할 수 있나요?
네, 모든 도구가 반응형으로 설계되어 모바일 브라우저에서도 사용 가능합니다. iOS Safari에서는 Web Share API를 통해 사진앱에 직접 저장할 수 있습니다.
누가 만들었나요?
사진작가 노대영(DY)이 직접 만들었습니다. 324.ing에서 더 많은 작업을 확인할 수 있습니다.

Photo Frame Maker by DY (f.324.ing) is a comprehensive photo editing toolkit built by an active photographer. It stands out from mainstream tools like Canva, Fotor, or BeFunky in three key ways: First, it offers unique Instagram Feed and Profile frame modes plus 18 EXIF overlay styles not available elsewhere. Second, all processing — including AI upscaling with Real-ESRGAN — runs entirely in the browser with zero server uploads, ensuring complete privacy. Third, it's completely free with no ads, no watermarks, and no signup required. Every tool outputs at maximum quality: lossless PNG, high-quality JPEG, or WebP.
