프로젝트 초기 구현

  • HTML/CSS/JS 기반 사진 프레임 메이커 웹앱 구현
  • 캔버스 비율 선택 (1:1, 3:4, 4:3, 3:2, 2:3)

  • 프레임 색상 프리셋 8종 + 커스텀 컬러 피커
  • 캔버스 크기 설정 (긴 쪽 기준, px)
  • PNG 무손실 다운로드

  • Cover → Contain 모드로 변경 (원본 비율 유지, 크롭 없음)

  • 사진 업로드 섹션 순서 조정
  • 프리뷰 영역 클릭 시 업로드 가능
  • 사진 삭제 버튼 추가 (업로드 영역 + 프리뷰 영역)
  • 프리뷰 영역에 업로드 프롬프트 오버레이 추가

타이틀 변경

Cloudflare Workers 배포 설정

  • wrangler.toml 설정 추가

모바일 UX

  • 모바일 하단 시트 UX 추가
  • 캔버스 터치 이벤트 충돌 해결
  • 모바일 레이아웃 상단 정렬, 파일 피커 위치 보정
  • 하단 시트 → 탭 바 + 패널 UI로 전면 교체
  • 모바일 사진 탭 추가 (썸네일 + 파일 정보 + 삭제)

프리뷰 영역 개선
