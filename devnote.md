# Dev Note — Photo Frame Maker by 324.ing

## 2026-02-27

### 프로젝트 초기 구현
- HTML/CSS/JS 기반 사진 프레임 메이커 웹앱 구현
- 캔버스 비율 선택 (1:1, 3:4, 4:3, 3:2, 2:3)
- 프레임 두께 조절 (슬라이더 0~25%)
- 프레임 색상 프리셋 8종 + 커스텀 컬러 피커
- 캔버스 크기 설정 (긴 쪽 기준, px)
- PNG 무손실 다운로드

### 이미지 표시 모드 변경
- Cover → Contain 모드로 변경 (원본 비율 유지, 크롭 없음)

### UI 레이아웃 조정
- 사이드바를 우측 패널로 배치
- 사진 업로드 섹션 순서 조정
- 프리뷰 영역 클릭 시 업로드 가능
- 사진 삭제 버튼 추가 (업로드 영역 + 프리뷰 영역)
- 프리뷰 영역에 업로드 프롬프트 오버레이 추가

### 타이틀 변경
- 헤더 및 페이지 타이틀을 "Photo Frame Maker by 324.ing"으로 변경

### Cloudflare Workers 배포 설정
- `wrangler.toml` 설정 추가

### 모바일 UX
- 모바일 하단 시트 UX 추가
- 캔버스 터치 이벤트 충돌 해결
- 모바일 레이아웃 상단 정렬, 파일 피커 위치 보정
- 하단 시트 → 탭 바 + 패널 UI로 전면 교체
- 모바일 사진 탭 추가 (썸네일 + 파일 정보 + 삭제)

### EXIF 메타데이터
- JPEG EXIF 파싱 구현 (카메라, 렌즈, 초점거리, 조리개, 셔터속도, ISO, 촬영일, 소프트웨어)
- 정보 섹션에 EXIF 데이터 표시

### 다운로드 개선
- 파일명을 `{원본이름}_pfm.png` 형식으로 변경
- iOS Web Share API 연동 (사진 앱에 저장)
- Web Share 미지원 시 길게 눌러 저장 오버레이 폴백
- User gesture 보존 처리

### 프리뷰 영역 개선
- 체크무늬 배경을 캔버스 비율에 맞게 표시
- JS 기반 프리뷰 컨테이너 크기 계산
